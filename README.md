# voicevue-agent

AI interview agent app with Next.js, Prisma, and voice-driven interview flows.

## Stack

Next.js, Prisma, TypeScript

## Getting started

```bash
git clone https://github.com/ANSHSINGH050404/voicevue-agent.git
cd voicevue-agent
bun install
bun run dev   # or: bun run start
```
